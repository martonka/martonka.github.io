org.kududb.flume.sink

Interface KuduEventProducer

  • All Superinterfaces:
    org.apache.flume.conf.Configurable, org.apache.flume.conf.ConfigurableComponent

    @InterfaceAudience.Public
@InterfaceStability.Evolving
public interface KuduEventProducer
extends org.apache.flume.conf.Configurable, org.apache.flume.conf.ConfigurableComponent
    Interface for an event producer which produces Kudu Operations to write the headers and body of an event in a Kudu table. This is configurable, so any config params required should be taken through this. The columns should exist in the table specified in the configuration for the KuduSink.
