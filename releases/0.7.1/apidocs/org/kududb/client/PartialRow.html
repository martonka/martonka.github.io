org.kududb.client

Class PartialRow

  • java.lang.Object
    • org.kududb.client.PartialRow

  • @InterfaceAudience.Public
@InterfaceStability.Evolving
public class PartialRow
extends Object
    Class used to represent parts of a row along with its schema.

    Values can be replaced as often as needed, but once the enclosing Operation is applied then they cannot be changed again. This means that a PartialRow cannot be reused.

    Each PartialRow is backed by an byte array where all the cells (except strings and binary data) are written. The others are kept in a List.

    This class isn't thread-safe.
